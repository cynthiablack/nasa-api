How to use this site

This site queries NASA's Mars Rover program image database. To use, enter a date in the search box,, then click the name of the Rover you'd like images from. If an image for the date you entered exists for that Rover in NASA's public database, it will appear on the page, below. If multiple images appear, they will appear in a carousel; you will need to click the left and right arrows to see all of the images.
